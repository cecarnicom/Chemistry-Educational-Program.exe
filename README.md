Program Name: Organic Chemistry Assistant Platform: Windows (Native Win32/64 - Single Executable) Author: Carnicom Institute

About this Program: This is an educational tool designed to assist in the study of chemistry/organic chemistry. It is an graphic interface application that runs efficiently on Windows devices, including tablets with limited resources.

Connectivity & Privacy:

Internet Usage: This program is designed to work entirely offline. It does not require an internet connection for any core functions.

Optional Features: The program includes optional links to external educational resources for additional assistance. These are entirely elective and only accessed if the user chooses.

Data Privacy: This tool does not collect, store, or transmit user data.

Terms of Use & Support:

Cost: Provided free of charge for educational use.

Support: Provided "as-is." There is no technical support or maintenance.
